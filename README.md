# HiLife Task 2 - Responsive Landing Page

This project is a responsive SaaS landing page developed as part of an internship assignment.

## Project Objective

The goal of this assignment was to recreate a modern business landing page using:

- HTML5
- CSS3
- Bootstrap
- JavaScript

The landing page was designed based on a provided reference UI and made fully responsive for different screen sizes.

---

## Features

- Responsive Navbar
- Hero Section
- Demo Request Form
- Feature Sections
- Channel Cards
- Statistics Cards
- Download Playbook Section
- Informational Cards
- Responsive Design
- Hover Effects
- Modern SaaS UI Layout

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
